# Classificador de Nível de Herói

Este é um programa em **JavaScript** que classifica o nível de um herói baseado na quantidade de XP informada. O objetivo é demonstrar o uso de variáveis, operadores, estruturas de decisão e laços de repetição de forma prática.

---

## Pré-requisitos

- **Node.js** instalado na sua máquina.  
  [Baixe aqui](https://nodejs.org/)

- Pacote `prompt-sync` para capturar entradas do usuário no terminal.

---

## Instalação e Execução

1. Clone este repositório ou baixe o arquivo `heroi.js`.
2. Abra o terminal na pasta onde está o arquivo.
3. Instale o pacote `prompt-sync`:

```bash
npm install prompt-sync
Execute o programa:

```bash
node heroi.js
Insira os dados solicitados:
Nome do herói
Quantidade de XP

O programa exibirá a mensagem final, por exemplo:

```bash
O Herói de nome Arthur está no nível de Ouro
